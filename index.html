ALL CSS SELECTOR

Selector Description Example
tag Selects all elements of a specific type. button { background-color: pink; }
.class Selects all elements with a specific class. .highlight { font-weight: bold; }
#id Selects a specific element with a unique ID. #main-header { font-size: 36px; }
parent > child Selects all direct child elements of a specific parent. header > h1 { color: red; }
element1 + element2 Selects the next element immediately following a specific element. p + a { text-decoration: underline; }
element1 ~ element2 Selects all elements following a specific element. h1 ~ p { font-style: italic; }
[attribute] Selects all elements with a specific attribute. [title] { border: 1px solid black; }
[attribute="value"] Selects all elements with a specific attribute and value. [type="text"] { padding: 5px; }
:pseudo-class Selects elements based on a specific state or condition. a:hover { color: red; }
